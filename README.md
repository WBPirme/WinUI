# OGAS - 桌面端生产资源管理与计划分配系统

[![.NET 8](https://img.shields.io/badge/.NET-8.0-purple.svg)](https://dotnet.microsoft.com/)
[![WPF](https://img.shields.io/badge/UI-WPF-blue.svg)](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/)
[![EF Core](https://img.shields.io/badge/ORM-EF_Core-green.svg)](https://learn.microsoft.com/en-us/ef/core/)

OGAS 是一个基于 `WPF + .NET 8 + EF Core + SQL Server LocalDB` 构建的桌面端生产资源管理与计划分配系统。

它不是一个单纯的“数据录入工具”，而是一个**面向生产管理场景的综合业务系统原型**。项目围绕生产型业务的核心环节展开，将基础资料维护、生产计划管理、生产执行、库存监控、出口订单以及首页数据看板整合到同一个应用中，形成了一条完整且闭环的业务流程链路。

##  核心亮点

- **端到端业务联动**：覆盖从“配置资源”到“查看产出与收益”的全生命周期，各个模块的数据结果相互影响。
- **动态资源调度**：带有基于业务约束的计划分配与执行推进逻辑，如基于设备能力和工厂效率自动计算生产时间。
- **现代化桌面架构**：采用 `WPF + MVVM` 模式，结合 `EF Core` 和依赖注入，结构清晰，易于扩展。
- **可视化数据总览**：首页提供直观的看板，集中展示计划状态、库存情况、出口收益与系统日志。

##  适用场景

##  业务流程全景

系统将生产活动中最常见的环节紧密串联：

`基础数据维护` ➔ `生产计划制定` ➔ `自动生产处理` ➔ `库存自动变动` ➔ `出口订单结算` ➔ `数据汇总展示`

1.  **配置基础资源**：定义产品、材料、设备、工厂和工艺（生产规则模板）。
2.  **创建生产计划**：系统根据工艺要求自动匹配合适的工厂资源，生成待执行任务。
3.  **启动生产处理**：引擎读取未完成计划，结合各项效率参数模拟生产进度与等待状态。
4.  **同步库存与记录**：任务完成后，自动扣减材料、增加成品，并生成生产记录。
5.  **出口与收益结算**：创建出口订单时校验库存，完成出库并自动计算业务收益。

## 核心功能模块

### 1. 首页主控台 (Dashboard)
系统的数据中枢，集中展示计划状态分布、实时库存水位、出口总收益、网络流量图表以及系统运行日志。

### 2. 基础资料管理 (Master Data)
支持包含关联约束的主数据维护：
* **产品与材料**：定义基本属性与计量单位。
* **生产设备与加工厂**：绑定工厂与设备能力，设定容量与效率。
* **生产工艺**：定义“生产规则”，关联产品、耗材、所需设备及标准工时。

### 3. 生产计划与执行 (Planning & Execution)
* **计划分配**：根据工艺自动筛选满足设备条件的加工厂，确保计划的业务合理性。
* **自动生产**：开启后自动遍历并推进任务，支持等待、暂停与恢复，实时展示进度条和剩余时间。

### 4. 库存与出口管理 (Inventory & Export)
* **库存监控**：独立管理产品库与材料库，支持手动调整与自动同步。
* **出口结算**：选择产品出库，自动校验库存余量并根据单价核算最终收益。

## 技术栈与架构

* **运行环境**: `.NET 8` (`net8.0-windows`)
* **UI 框架**: WPF
* **架构设计**: MVVM (Model-View-ViewModel)
* **数据访问**: Entity Framework Core
* **数据库**: SQL Server LocalDB
* **依赖注入**: `Microsoft.Extensions.Hosting`
* **图表组件**: OxyPlot
* **日志记录**: NLog

**代码目录结构职责：**
* `Views`：界面页面文件 (XAML)
* `ViewModels`：页面逻辑与业务处理
* `Models`：核心业务实体建模
* `Data`：数据库上下文与数据访问入口
* `Services`：导航、弹窗等通用服务
* `Controls`：自定义控件
* `Converters`：数据绑定转换器
